# Angular portfolio

### Usage


### Run dev server

```sh
$ npm install http-server -g
```

```sh
$ http-server src -c-1
```

### In new shell window run

```sh
$ grunt
```

### To build production package run

```sh
$ grunt build
```

### To preview production build run

```sh
$ grunt build
```

```sh
$ http-server dist
```

### All production files are in dist folder
